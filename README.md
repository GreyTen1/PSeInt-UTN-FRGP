# PSeInt-UTN-FRGP

Una version modificada del PSeInt con los simbolos que la carrera TUP (de la UTN-FRGP) promueve.

Para ejecutar el programa simplemente basta con ir al release y bajarse la version que necesitan.

⚠️ Atencion ⚠️

Esta todavia en fase de prueba, el unico binario disponible es el de Linux

(para extraerlo, simplemente ejecuten "tar -xvf nombre_del_comprimido.tar.gz")

ejecutando "./pseint" les deberia iniciar el programa
